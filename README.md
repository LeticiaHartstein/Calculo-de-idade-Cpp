# Calculo-de-idade-Cpp
Um simples código de cálculo de idade em C++
<br>
<h2>Fluxograma</h2>
<img src="https://raw.githubusercontent.com/LeticiaHartstein/Calculo-de-idade-C-/refs/heads/main/Fluxograma-CalculoDeIdade.svg" width="500" alt="Fluxograma">
<h2>Explicação/ Lógica: </h2>
<ul>
  <li>Armazenar os dados do dia, mes e ano separados</li>
  <li>Subtrair o ano atual com o ano de nascimento para ter a idade</li>
  <li>Verificar se o mês atual é menor que o de nascimento</li>
  <li>Verificar se o mês atual é igual ao de nascimento e se o dia atual é menor que o de nascimento</li>
  <li>Se o mes satisfazer qualquer uma das duas condições, o programa subtrai um ano da idade</li>
</ul>
